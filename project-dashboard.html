Java Scheduler

An individual project utilizing Java and Object-Oriented Programming to develop a calendar software for students. The software handled importing of class schedules and due dates, saving and recalling schedules, and differentiating between types of events with color-coding.

Roy G. Biv

A 4-person team project where I acted as a Scrum Master and Developer. The goal of the project was to aid those with color blindness to give them a better user experience. Using Python, a desktop software was developed that allowed the user to filter their screen based off their preference to help differentiate colors.

Book Nook

An individual project utilizing Java and a SQL database. The goal of the project was to design a database for readers and authors to allow them to store information about books and reviews and integrate that with an user-interface platform. It allowed for personalized profiles for each type of user.
